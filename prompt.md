# Задача

Собери полную HTML-страницу events для сайта icoda.io. Это маркетинговое агентство, страница — календарь крипто/AI/iGaming конференций.

## Исходные файлы

- `mockups/` — скриншоты текущего сайта icoda.io/events/ по блокам. Страницу нужно повторить максимально близко к этим скринам визуально.
- `reference.html` — черновой макет с правильной структурой, логикой и данными. Используй как основу для контента и интерактива.
- `brief.md` — описание изменений, которые нужно внести относительно текущего сайта.

## Что сделать

Собери один HTML-файл (`events.html`) со встроенными стилями и скриптами. Страница должна:

1. Визуально максимально повторять скриншоты из `mockups/`
2. Включать все изменения из `brief.md`
3. Быть полностью рабочей — hover на hero меняет баннер, табы фильтруют, Show more раскрывает карточки, промокоды копируются, бегущие строки партнёров анимируются

## Структура страницы сверху вниз

1. **Nav** — шапка сайта (повторить по скрину)
2. **Hero** — 3 ближайших конференции, hover меняет баннер справа. Кнопки: Grab your spot, Read our review (если есть блог-пост), промокод + Copy (если есть)
3. **Featured events** — табы (All / Crypto & Web3 / AI & Tech / iGaming), фильтры, карточки ивентов. Изначально 3 карточки, Show more раскрывает ещё 3
4. **Meet ICODA worldwide** — повторить по скрину
5. **Our partners** — две бегущие строки с лого конференций. Верхняя строка справа налево, нижняя слева направо. Loop-анимация, hover ставит на паузу
6. **You may like** — повторить по скрину
7. **Footer** — повторить по скрину

## Стиль

- Тёмная тема (background ~#0B0B1E)
- Акцентный цвет: #5B50FF (фиолетово-синий)
- Шрифт: Inter
- Карточки: background ~#111128, скруглённые углы 12-16px
- Минималистичные бордеры rgba(255,255,255,0.06)

## Важно

- Один файл, без внешних зависимостей кроме Google Fonts
- Мобильная адаптивность не нужна на этом этапе
- Если на скрине видны конкретные отступы, размеры шрифтов, цвета — повторяй их точно
- Каждый блок бери из скрина и подгоняй pixel-perfect
